# Cadastro de usuários

Um sistema web que registra informações de usuários e as retorna.

## Tecnologias

- [React](https://reactjs.org/)
- [TailwindCSS](https://tailwindcss.com/)
- [Vite](https://vitejs.dev/)

## Instlação

Clone o repositório e execute localmente

```bash
git clone https://github.com/mateusmarques-B/cadastro-usuarios
cd cadastro-usuarios
npm install
npm run dev
```
